# Code of Conduct

:::caution Draft / TBD
A formal Code of Conduct, enforcement process and reporting channel have not been approved for publication.
:::

## Draft principles

- Treat contributors, learners, competitors and researchers with respect.
- Do not use community spaces to share private data, credentials or unsafe operating instructions.
- Keep technical disagreement focused on evidence, reproducibility and the relevant project scope.

## To be confirmed

- Formal policy text and enforcement roles.
- Reporting channel, confidentiality and escalation process.
- Scope across repositories, events and communication spaces.
