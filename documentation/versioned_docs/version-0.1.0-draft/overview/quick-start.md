# Quick Start

:::caution Draft / TBD
No installation command, supported host platform, firmware image, wiring procedure or release package has been approved for publication yet.
:::

The future quick-start will lead a new user through a minimal, versioned path:

1. Select a documented Zeno platform and release version.
2. Review safety, required equipment and recovery procedures.
3. Install only the matching software and firmware artifacts.
4. Validate connectivity before enabling motion or mission behavior.
5. Record the platform, software and simulation versions used.

Until those prerequisites are published, do not treat this page as an operating procedure.
