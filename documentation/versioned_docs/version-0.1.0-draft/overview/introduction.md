# Introduction

:::caution Draft / TBD
This documentation site is an information architecture draft. It does not yet publish verified hardware specifications, compatibility matrices, release artifacts or support commitments.
:::

Zeno is a full-stack open-source embodied intelligence platform for education, research and robotics competition.

The documentation is organized so learners and developers can move from system context to hardware, firmware, software, simulation, robot platforms and community governance without treating those layers as unrelated projects.

## What this site will contain

- Versioned quick-starts, concepts, tutorials, how-to guides and references.
- Explicit scope and version labels for every technical claim.
- Links to authoritative product repositories once their public locations are confirmed.
