# Architecture

:::caution Draft / TBD
This is a platform information model, not a published compatibility guarantee or implementation specification.
:::

Zeno organizes embodied intelligence across connected layers:

1. **Zeno Link** for device, bus and communication integration.
2. **Zeno Core** for shared runtime, foundational APIs and core interfaces.
3. **Zeno Cerebellum** for real-time motion control and execution.
4. **Zeno Brain** for perception, planning and intelligent task orchestration.
5. **Zeno Humanoid** and **Zeno Marine** for robot platform contexts.
6. **Zeno Simulation** and optional **Zeno Cloud** for testing and collaboration workflows.

## Documentation rule

Each future page must state which layer it addresses, the applicable version and any required hardware or software dependencies. Do not infer compatibility across layers without a published matrix.
