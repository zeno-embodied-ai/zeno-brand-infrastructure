# ADR 0005: Zeno adopts an original geometric logo asset

- Status: accepted
- Date: 2026-07-14

## Context

The website previously used a temporary CSS/SVG mark. Zeno needs one compact visual identifier for the website and GitHub Organization profile.

## Decision

Adopt `website/public/brand/zeno-logo.png` as the initial Zeno logo asset. The mark is an original three-segment geometric Z: a single cyan diagonal framed by two deep-navy strokes. This deliberately minimal silhouette expresses forward motion and a connected open platform without decorative joints or orbital elements. It is a square RGBA PNG and is used by the website header, footer, metadata and web-app manifest.

## Consequences

- The asset is suitable for square/circular avatar contexts and does not include a wordmark.
- This decision authorizes this repository's website use. Trademark clearance, formal visual-system specification and replacement of the Open Graph placeholder remain separate Brand/Legal work.
- The GitHub Organization avatar must use the identical PNG through GitHub's organization-profile controls; changing it does not change Organization ownership, billing, membership or repository permissions.
